1.¿Qué es un método (en Javascript)?

Es una función que está asociada a un objeto o clase. Se define dentro de un objeto o clase y puede acceder a las propiedades del mismo.

const persona = { nombre: "Juan", saludar() { console.log("Hola, " + this.nombre); } };

2. Constructor de clase

Es un método especial en una clase que se ejecuta automáticamente cuando se crea una instancia de la clase. Se usa para inicializar los valores de las propiedades del objeto.

class Persona { constructor(nombre, edad) { this.nombre = nombre; this.edad = edad; } }

3. Método estático

Es un método que pertenece a la clase en sí, no a las instancias de la clase. Se accede a él a través de la clase y no requiere crear una instancia.

class MathUtils { static sumar(a, b) { return a + b; } }

4. Métodos públicos

Son métodos accesibles desde fuera de la clase o el objeto. Se pueden invocar en cualquier momento después de crear la instancia.

class Persona { saludar() { console.log("Hola"); } }

5. Métodos privados

on métodos accesibles solo dentro de la propia clase. En JavaScript moderno, se definen usando #.

class Persona { #saludar() { console.log("Hola"); } }

6. Herencia de clases

Es el mecanismo que permite que una clase herede las propiedades y métodos de otra clase. Esto permite la reutilización de código y una jerarquía de clases.

class Animal { caminar() { console.log("El animal camina"); } } class Perro extends Animal { ladrar() { console.log("Guau"); } }
